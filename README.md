## 🛠 Capstone Design Project

### Cross-Predictive Reverse Distillation Based Anomaly Detection Framework for Multimodal Industrial Anomaly Detection

**2026 Smart Factory Convergence Capstone Design 1 · Team 9** (Partner company: Duplex)

> A memory-bank-free multimodal anomaly detection framework that jointly leverages RGB images and surface normal maps to detect both appearance and geometric defects. Cross-modal reverse reconstruction mitigates the anomaly leakage problem common in prior distillation-based methods.

- **Team:** Dongwoo Kim (Lead) · Minseok Kang · Minseung Kook · **Yoonkyeong Na** · Dongseon Lee

- **My Role:** Related work research & organization, demo model interface implementation, presentation panel design

- **Key Results:** I-AUROC 96.85% on MVTec 3D-AD, 93.31% on Eyecandies, 45.0 FPS / 535MB Peak VRAM without a memory bank
